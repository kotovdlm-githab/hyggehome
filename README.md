# hyggehome — сайт (статическая версия)

Готовый статический сайт: HTML + CSS + JS, без сборки и без сервера. Открывается на любом хостинге.

## Страницы
- `index.html` — главная (герой с фото леса, категории, о бренде, ценности)
- `catalog.html` — каталог: категории → подкатегории + поиск + сортировка
- `product.html` — карточка товара (`product.html?id=k01`)
- `cart.html` — корзина, скидка лояльности, доставка, оформление
- `account.html` — регистрация/вход, лояльность, история заказов
- `404.html`, `robots.txt`, `sitemap.xml`, `seo/semantic-core.md`

## Где всё настраивается
Файл `assets/js/store.js`, объект `HH_CONFIG` в начале:
- `social.vk / telegram / instagram` — ссылки на соцсети (шапка и подвал)
- `marketplaces.wildberries / ozon` — витрины на маркетплейсах
- `contacts.email / phone` — контакты в подвале
- `telegramBot` — имя бота (вход/уведомления)
- `loyalty.tiers` — уровни и скидки лояльности
- `delivery.methods` / `delivery.freeFrom` — способы и порог бесплатной доставки
- `paymentProvider` — провайдер оплаты
- `apiBase` — адрес бэкенда (пусто = демо-режим на localStorage)

## Режим работы
Сейчас корзина, аккаунты и заказы хранятся в браузере (localStorage) — это демо.
Для «боевого» режима (реальная оплата, Telegram-бот, живые тарифы доставки, серверные аккаунты)
нужен бэкенд; точки подключения размечены в коде комментариями «ИНТЕГРАЦИЯ».
